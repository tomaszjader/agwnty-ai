# Kurs Agentów AI dla Programistów

Landing page promujący kurs tworzenia agentów AI z wykorzystaniem:

- MCP (Model Context Protocol)
- Tool Calling
- RAG (Retrieval Augmented Generation)
- Integracji API
- Vector Databases
- LLM (OpenAI, Anthropic, Gemini, Ollama)

## O projekcie

To prosta strona marketingowa napisana w jednym pliku HTML.

Celem strony jest prezentacja kursu uczącego budowy produkcyjnych agentów AI dla programistów.
