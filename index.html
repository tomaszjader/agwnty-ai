Czego się nauczysz?

Program prowadzi od podstaw pracy z modelami językowymi do agentów połączonych z narzędziami, bazami wiedzy i aplikacjami.

M0

Promptowanie i fundamenty LLM

Jak rozmawiać z modelem, ustawiać kontekst, testować odpowiedzi i myśleć o ograniczeniach AI.

M1

Architektura agentów AI

Planning, pamięć, workflows, pętle decyzyjne i różnice między agentem a zwykłym chatbotem.

M2

Tool calling

Projektowanie narzędzi, JSON Schema, walidacja danych wejściowych i obsługa błędów.

M3

MCP

Model Context Protocol, własne MCP serwery i podłączanie agentów do realnych aplikacji.

M4

RAG

Embeddings, chunking, wyszukiwanie semantyczne, reranking i odpowiedzi oparte o dokumenty.

M5

Integracje i wdrożenie

API, GitHub, bazy danych, dokumentacja firmowa, bezpieczeństwo, monitoring i deploy.

Co zbudujesz po drodze?

Kurs jest praktyczny: każdy temat prowadzi do umiejętności, które możesz wykorzystać we własnej aplikacji albo projekcie firmowym.

Agent do pracy z dokumentacją

Asystent odpowiadający na podstawie PDF-ów, Notion, Confluence albo wewnętrznej bazy wiedzy.

Agent korzystający z API

System, który pobiera dane, uruchamia akcje i zwraca wynik w kontrolowanym formacie.

Agent z własnymi narzędziami MCP

Integracja z lokalnymi lub firmowymi narzędziami, które agent może wywoływać w bezpieczny sposób.

Po kursie potrafisz

  • Projektować agenta AI jako proces, a nie tylko okno czatu.
  • Tworzyć narzędzia, które model może wywoływać przez tool calling.
  • Budować RAG dla dokumentacji, kodu i baz wiedzy.
  • Łączyć agenta z API, bazą danych i systemami firmowymi.
  • Rozumieć ryzyka: prompt injection, uprawnienia i błędne akcje.
  • Przygotować rozwiązanie do testowania, monitoringu i wdrożenia.
